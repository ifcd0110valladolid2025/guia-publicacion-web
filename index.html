Módulo MF0952_2 - Publicación de páginas web

Introducción

Este módulo formativo te enseña los fundamentos de la publicación de páginas web, incluyendo las mejores prácticas para subir contenido a servidores, optimizar para motores de búsqueda y gestionar archivos web.

Objetivos de aprendizaje
  • Configurar y usar clientes FTP/SFTP como FileZilla
  • Generar archivos sitemap.xml para SEO
  • Crear archivos robots.txt para control de indexación
  • Optimizar meta etiquetas para motores de búsqueda
  • Dominar comandos básicos de Windows y Linux

FTP/SFTP y FileZilla

Diferencias entre FTP y SFTP

Característica FTP SFTP
Seguridad Sin cifrado Cifrado SSH
Puerto 21 22
Autenticación Usuario/contraseña SSH keys o usuario/contraseña
Velocidad Rápido Ligeramente más lento
Recomendación: Siempre usa SFTP cuando sea posible por seguridad.

Configuración de FileZilla

Paso a paso:
  1. Descarga FileZilla desde filezilla-project.org
  2. Abre el Gestor de Sitios (Ctrl+S)
  3. Haz clic en "Sitio nuevo"
  4. Configura los datos del servidor:
    • Servidor: tu-servidor.com
    • Protocolo: SFTP
    • Puerto: 22
    • Usuario: tu-usuario
    • Contraseña: tu-contraseña
  5. Haz clic en "Conectar"

Resolución de errores comunes

Causa: Permisos insuficientes en archivos o carpetas.
Solución: Cambia los permisos a 755 para carpetas y 644 para archivos.

Causa: Archivo no existe o ruta incorrecta.
Solución: Verifica que el archivo index.html esté en la carpeta correcta.

¿Qué es un sitemap?

  • Ayuda a los motores de búsqueda a entender la estructura de tu sitio.
  • Incluye detalles como la última fecha de actualización y la prioridad de cada página.

Generador de Robots.txt

¿Qué es robots.txt?
  • Controla el acceso de bots a tu sitio web.
  • Previene la indexación de contenido duplicado o privado.
  • Mejora la eficiencia del rastreo de los motores de búsqueda.
  • User-agent: especifica el bot al que se aplican las reglas (ej. *, Googlebot).
  • Disallow: bloquea rutas o páginas específicas.
  • Allow: permite el rastreo de rutas dentro de secciones bloqueadas (útil con Googlebot).
  • Sitemap: indica la URL del archivo sitemap.xml para facilitar la indexación.

Actividades Prácticas

Ejercicio 1: Configurar conexión SFTP

Aprende a configurar FileZilla para conectarte a un servidor.

  1. Abre FileZilla
  2. Accede al Gestor de Sitios
  3. Crea un nuevo sitio con protocolo SFTP
  4. Configura los datos de conexión
  5. Realiza la primera conexión
Ejercicio 2: Crear sitemap para proyecto local

Genera un sitemap.xml para tu proyecto de desarrollo.

  1. Identifica las páginas de tu proyecto
  2. Añade las URLs usando el generador
  3. Configura prioridades y frecuencias
  4. Genera y descarga el sitemap.xml
  5. Coloca el archivo en la raíz del proyecto
Ejercicio 3: Configurar robots.txt para blog

Crea un archivo robots.txt que proteja áreas sensibles.

  1. Identifica qué páginas no deben indexarse
  2. Usa el generador para crear reglas
  3. Añade la URL del sitemap
  4. Genera y descarga robots.txt
  5. Sube el archivo a la raíz del servidor
Ejercicio 4: Optimizar meta etiquetas

Crea meta etiquetas optimizadas para SEO.

  1. Escribe un título atractivo (máx. 60 caracteres)
  2. Redacta una descripción convincente
  3. Configura Open Graph para redes sociales
  4. Añade Twitter Cards
  5. Implementa las etiquetas en tu HTML
